1. Menjelaskan fungsi modul
   Modul adalah bagian-bagian dari aplikasi yang punya tugas masing-masing, supaya program lebih rapi dan mudah dipahami.

2. Menjelaskan alur Frontend → Backend → Database
   Frontend menampilkan tampilan ke pengguna, Backend memproses data, dan Database menyimpan data.

3. Menjelaskan loading state dan error handling
   Loading state digunakan untuk menampilkan proses sedang berjalan, sedangkan error handling untuk menampilkan pesan jika terjadi kesalahan.

4. Menjelaskan fungsi GET, POST, PUT, dan DELETE
  GET: mengambil data
  POST: menambahkan data
  PUT: mengubah data
  DELETE: menghapus data
